## GanitSūtram Internationalisation
Author: Jawahar R Mallah | AITDL
VS 2082 | 2026-03-07

## Supported Languages
| Code | Language  | Script     | Status   |
|------|-----------|------------|----------|
| en   | English   | Latin      | Complete |
| hi   | Hindi     | Devanagari | Complete |
| sa   | Sanskrit  | Devanagari | Partial  |

## Architecture
- Locale files: /ui-core/locales/{locale}.json
- Engine: GanitI18n (vanilla JS, no library)
- Storage: localStorage ('gs_locale')
- DOM: data-i18n attributes

## Adding a New Language
1. Create websites/ui-core/locales/{code}.json
2. Add locale code to GanitI18n.getAvailableLocales()
3. Add button to .gs-lang-switcher in nav
4. Add font support in i18n.css if needed

## Key Conventions
- Nested dot notation: 'nav.home', 'gate.personas.student'
- Variable interpolation: "Hello {{name}}"
- Fallback chain: requested → en → key string
- Sanskrit invocations: never translated (language-independent)
