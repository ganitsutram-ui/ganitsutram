# GaṇitSūtram

**The Golden Book of Vedic Mathematics**

Official website and digital platform for **GaṇitSūtram**, a self-learning journey into Vedic Mathematics designed to help students understand mathematical patterns clearly and naturally.

---

## About the Project

GaṇitSūtram is a modern learning initiative inspired by the mathematical traditions of **Ancient Bharat**.
The project aims to make mathematics **simpler, faster, and more intuitive** through pattern-based thinking rather than memorizing long procedures.

This repository contains the source code for the **official GaṇitSūtram website and learning platform**.

The platform will gradually expand to include:

* Interactive Vedic Mathematics tools
* Daily math challenges
* Step-by-step learning modules
* AI-assisted math explanations
* Student progress tracking
* Educational resources aligned with school curricula
